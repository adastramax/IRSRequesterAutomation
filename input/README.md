# QA Bulk Test Inputs

These CSVs are for bulk QA testing only.

- `bulk_add_mixed_english_01.csv` covers mixed add cases with blank `Site ID`, provided `Site ID`, and new-site candidates.
- `bulk_add_mixed_english_02.csv` covers another mixed add set with different names and sites.
- `bulk_add_mixed_english_03.csv` covers a third mixed add set for smoke testing.
- `bulk_deactivate_english.csv` covers bulk deactivate testing with valid required fields.
